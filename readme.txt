This plugin blocks text selections and right clicks on your blog pages and posts.

== Description ==

This plugin gives you an option to make your content safe. There is nothing like full protection of your content being copied. But this plugin does enough to make the content hacker work a little bit more for your content.

== Installation ==

1. Upload unzipped plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Plugin Options on left menu under Settings > JCWP copy paste blocker.
